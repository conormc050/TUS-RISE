# Python prototype

This folder contains a minimal prototype for posture/angle calculations.

How to run:
1. Create virtual env:
   python3 -m venv .venv
   source .venv/bin/activate
2. Install dependencies:
   pip install -r requirements.txt
3. Run prototype:
   python main.py

Notes:
- Replace main.py with your CV pipeline (Mediapipe, OpenCV, PyTorch, etc.)
- Consider adding Jupyter notebooks for dataset exploration.
